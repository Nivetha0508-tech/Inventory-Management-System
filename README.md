# 📦 Inventory Management System

A professional **Inventory Management System** developed using **C# Windows Forms**, **.NET**, and **SQL Server**. This application helps businesses efficiently manage products, sales, inventory, and reports through an easy-to-use desktop interface.

---

## 📌 Features

### 🔐 Login System
- Secure login screen
- Username and password authentication

### 📦 Product Management
- Add new products
- Update product details
- Delete products
- Search products
- Auto-generated Product ID
- Product stock management

### 🛒 Sales & Billing
- Generate customer bills
- Select products from inventory
- Automatic stock deduction after each sale
- Sales history

### 📊 Reports
- Today's Sales Report
- Monthly Sales Report
- Top Selling Products
- Revenue tracking

### 📉 Inventory Management
- View available stock
- Low stock alerts
- Product quantity tracking

### 🏠 Dashboard
- User-friendly navigation
- Quick access to Products, Sales, Inventory, and Reports

- Users
- Products
- Sales

---

# 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/Nivetha0508-tech/Inventory-Management-System.git
```

2. Open the solution in **Visual Studio Community 2022**.

3. Restore NuGet packages if prompted.

4. Create the required database and tables in SQL Server.

5. Update the SQL Server connection string in the project.

6. Run the application.
